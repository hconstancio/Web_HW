Summary: Lattitude vs X


The purpose of this project is to identify the different climate behaivors experienced when the location is closed to the EQUATOR. The data set is retriebed from a WebSite called OpenWeatherMap via API. There are at least 500 different locations being pulled to get a significant statistical sample.

After getting the data via Pandas for Python, MatPlotLib tool was used to create a graphical representation (Plot) like the one below.
